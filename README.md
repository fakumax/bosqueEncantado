# Bosque Encantado - Invitación XV Años Sofía

Invitación digital interactiva para los XV años de Sofía, con temática de bosque encantado.

## Tech Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS 4
- Framer Motion
- SCSS
- Radix UI + Phosphor Icons

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
